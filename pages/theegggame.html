The Egg Game

  • Unity Engine
  • C#
  • Android

The Egg Game is my one of my first finished game. At that time, I wanted to complete one project from begining to end, and release it on Android.

Donwload →

The Main mechanic of The Egg Game is to reach the end of the levels without suffering any shocks!

A shock is define by the velocity of the egg when it is hitting an object. When this velocity is exceeding a defined value, it's game over!
